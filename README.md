# NBA-Analytics-Engine

A data-driven platform for player projections, team analysis, and statistical matchup insights.

The NBA Analytics Engine is a custom-built basketball analytics tool designed to evaluate player performance, generate probability-based stat projections, and compare teams using real statistical data.
